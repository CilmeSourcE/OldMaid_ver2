# OldMaid
ババ抜きをするプログラムです。  
## 概要
コンソール上で、プレイヤーがババ抜きを行います。  
プログラムの流れとしては、  
  
ゲーム準備（トランプ用意とシャッフル、プレイヤーにカードを配るなど）  
↓  
ゲーム開始  
↓  
残りのプレイ人数が一人になった時点で試合終了。  
  
みたいな感じです。
  
## 実行方法
OldMaidMain.javaを実行することで、ババ抜きが開始されます。
